# Simple-OpenAI-Agent
# 🧠 Simple OpenAI Agent

This is a simple AI chatbot built using the OpenAI SDK and Python. You can ask it any question, and it will respond intelligently using the selected language model.

## 🚀 Features

- Simple command-line interface
- AI responses powered by OpenRouter
- Easy to run and extend

## 🔧 Tools & Technologies Used

- Python 🐍
- OpenAI SDK
- OpenRouter API
- `.env` for secure API key management

## 📁 Files Included

- `agent.py` – Main chatbot script
- `requirements.txt` – Python dependencies
- `.gitignore` – To keep sensitive files safe
